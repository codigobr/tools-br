Principais características do Conversor de HTML em Jade

  • Converte rapidamente de forma segura HTML para Jade
  • Após a conversão de HTML em Jade use o menu Encadear com... para direcionar o resultado da conversão diretamente para outra ferramenta compatível com o conteúdo do editor.
  • Carregue um conteúdo HTML arrastando um arquivo diretamente para dentro do editor
  • Copie ou faça o download do conteúdo convertido em Jade, ou até mesmo o seu conteúdo original em HTML
  • O conteúdo do editor é gravado no histórico, ou seja, quando você acessar novamente o site, o seu conteúdo HTML e Jade ainda estará aqui.
